# 🍽️ Restaurant Management System

Proyek ini adalah sistem manajemen restoran berbasis **Java** yang dikembangkan secara kolaboratif oleh Group 5 untuk mempermudah pengelolaan operasional restoran secara terstruktur dan efisien.

## 🚀 Fitur Utama
* **Manajemen Menu:** Mengelola daftar makanan, minuman, dan harga secara dinamis.
* **Sistem Pemesanan:** Pencatatan dan pengelolaan transaksi pesanan pelanggan.
* **Pengembangan Modular:** Struktur proyek yang dibagi berdasarkan modul per mahasiswa untuk mempermudah integrasi dan kolaborasi tim.

## 📂 Struktur Proyek
```text
Group5-Restaurant-System/
├── .vscode/             # Konfigurasi workspace dan editor
├── shared/              # Resource, library, atau kelas yang digunakan bersama
├── student1_rusdi/      # Modul fitur khusus: Rusdi
├── Student2_Dzaki/      # Modul fitur khusus: Dzaki
├── student3-ahmad/      # Modul fitur khusus: Ahmad
└── student4_dinda/      # Modul fitur khusus: Dinda
```

## 🛠️ Persyaratan Sistem
* **Java Development Kit (JDK):** Versi 8 atau yang lebih baru.
* **IDE:** IntelliJ IDEA, NetBeans, Eclipse, atau VS Code (dengan ekstensi Java).
* **Git:** Untuk manajemen kontrol versi.

## ⚙️ Cara Menjalankan
1. **Clone repositori:**
   ```bash
   git clone https://github.com/dindanurhaliza/Group5-Restaurant-System.git
   ```
2. **Import Proyek:**
   Buka IDE Java Anda, pilih `Open Project`, lalu arahkan ke direktori `Group5-Restaurant-System`.
3. **Konfigurasi Build:**
   Pastikan *Project Structure* (JDK Path) sudah terkonfigurasi dengan benar di IDE Anda.
4. **Compile & Run:**
   Jalankan file utama yang berisi metode `public static void main(String[] args)` untuk memulai aplikasi.
