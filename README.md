# Restaurant Management System

Proyek ini adalah sistem manajemen restoran yang dikembangkan menggunakan bahasa pemrograman **Java**. Sistem ini dirancang untuk mengelola operasional restoran secara terstruktur.

## 🚀 Fitur Utama
* **Manajemen Menu:** Mengelola daftar makanan dan minuman.
* **Sistem Pemesanan:** Mengelola transaksi pesanan pelanggan.
* **Modul Terintegrasi:** Pengembangan dilakukan secara kolaboratif dengan pembagian modul per mahasiswa.

## 🛠️ Teknologi yang Digunakan
* **Bahasa Pemrograman:** Java
* **Tools:** IDE (IntelliJ IDEA, NetBeans, atau Eclipse), Git

## ⚙️ Cara Menjalankan
1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/dindanurhaliza/Group5-Restaurant-System.git
   ```
2. **Buka Proyek:**
   Buka folder proyek di IDE Java pilihan Anda.
3. **Konfigurasi:**
   Pastikan JDK (Java Development Kit) sudah terinstal di sistem Anda.
4. **Compile & Run:**
   Jalankan file utama (biasanya memiliki metode `public static void main`) untuk memulai aplikasi.
